# VORTEX – Sportswear E-Commerce Website
A modern, responsive e-commerce website for VORTEX Sportswear. Built as a portfolio project to demonstrate the capabilities of my custom React UI component library.

**DEMO**: https://vortex-e-commerce-7ldu.vercel.app/

# About

This project is a realistic e-commerce frontend for a sportswear store, designed with clean layouts, responsive grids, and an intuitive user experience. It showcases a shopping interface built entirely with my custom React component library `@yosang/react-ui`.

The goal was to create a highly interactive, frontend that highlights the UI library’s features, including Hero sections, ProductCards, flexible Containers, Drawers, Modals, and a sticky Navbar.

# Why I built it
- To test and showcase `@yosang/react-ui` in a e-commerce scenario.
- To practice building interactive, component-driven React applications.
- To create a visually appealing portfolio project with real-world functionality (shopping cart, login modal, product/brand pages).

# Features
- Fully responsive design for mobile, tablet, and desktop
- Hero section
- Sticky Navbar with multi-level navigation and icons
- Drawer system for sidebar menu and shopping cart
- Login modal with form inputs
- ProductCard component with discount, title, description, and price
- Brand showcase grid
- Smooth scrolling, modern typography, and clean layout

# Tech Stack
- React (Components, JSX, Router)
- React Router v6 for client-side routing
- [@yosang/react-ui](https://www.npmjs.com/package/@yosang/react-ui) – Custom component library
- Vite for fast builds and dev server
- CSS & Inline styles for fine-tuning layout
- Custom UI Components: Hero, ProductCard, Drawer, Modal, Navbar, Container, Input, Button, and more

### Installation & Usage

1. Clone the project with `https://github.com/yosang/vortex-e-commerce.git`
2. Navigate to the project directory with `vortex-e-commerce`
3. Installa required packaged with `npm install` 
4. Run the development environment with `npm run dev`
